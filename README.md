# 🌍 World News Trivia Game

A fun, shareable trivia game about the latest news happenings around the world. Test your knowledge, share your score with friends, and see how you rank on the leaderboard!

## ✨ Features

- **5 Daily Trivia Questions**: Questions rotate every 30 minutes to keep it fresh
- **AI-Powered Content**: Questions are generated from recent news using OpenAI API
- **Instant Feedback**: See your score immediately with correct answers highlighted
- **Share Your Score**: Generate a shareable link to compare scores with friends
- **Live Leaderboard**: See the last 50 players and their scores in real-time
- **Zero Installation**: Works in any browser - just visit the URL!
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
